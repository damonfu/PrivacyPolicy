允许程序访问设备电话状态"uses-permission android:name=android.permission.READ_PHONE_STATE获取设备唯一标识符，用于登录注册真实用户的依据、广告SDK形成特征标签，向您提供更多相关广告、收集统计应用产生的BUG信息，以便我们改进产品。
